## ✨ Características Técnicas

Este portafolio ha sido construido utilizando las tecnologías más modernas de la industria:

- **Framework:** [Next.js 15+](https://nextjs.org/) (App Router)
- **Estilos:** [Tailwind CSS v4](https://tailwindcss.com/)
- **UI Components:** [Shadcn UI](https://ui.shadcn.com/)
- **Iconografía:** [Lucide React](https://lucide.dev/)
- **Animaciones:** Framer Motion & CSS Animations (Efectos de flotación y fundido)
- **Diseño:** "Midnight Premium" (Negro Carbón y Azul Medianoche) con efectos de **Glassmorphism**.

## 🛠️ Instalación y Uso Local

Para ejecutar este proyecto en tu propia computadora, sigue estos pasos:

### 1. Clonar o descargar el proyecto
Si tienes git instalado:
```bash
git clone <url-del-repositorio>
cd portafolio
```

### 2. Instalar dependencias
He utilizado varias librerías de UI, por lo que este paso es crucial:
```bash
npm install
```

### 3. Iniciar el servidor de desarrollo
```bash
npm run dev
```
Luego, abre [http://localhost:3000](http://localhost:3000) en tu navegador para ver los resultados.

## 📂 Estructura del Proyecto

- `app/`: Contiene la lógica principal, rutas (`page.tsx`) y estilos globales (`globals.css`).
- `components/ui/`: Componentes base de Shadcn (Botones, Tarjetas, Badges).
- `public/`: Almacena recursos estáticos como tu foto de perfil (`profile.png`).
- `lib/`: Utilidades para el manejo de clases dinámicas.
